# How It Works: Aurora Parts Comparator

## Overview
This program helps sales users search, compare, and analyze solar parts and project designs efficiently. It connects to a backend API, fetches data, and displays results in a user-friendly interface.

## Main Features
- **Search Parts:** Find solar parts by SKU or name, filter by domestic/non-domestic, and view pricing.
- **Compare Projects:** View and compare multiple solar project designs, including pricing and component breakdowns.
- **Pagination:** Easily navigate large datasets with next/prev buttons and page controls.
- **Responsive UI:** Works well on desktop and mobile devices.

## How to Use
1. **Search for Parts:**
   - Enter a query (e.g., part name or SKU) and click 'Load'.
   - Results show key details: name, price, origin, etc.
2. **Browse Projects:**
   - Select the 'projects' endpoint and page through available projects.
   - Click on a project to view its designs and pricing.
3. **Compare Designs:**
   - Use the 'compare' endpoint to see detailed pricing and component analysis for each design.
4. **Pagination Controls:**
   - Use 'Next' and 'Prev' to move through pages of results.
   - For components, use the cursor for efficient navigation.

## Teaching Points
- **API Integration:** Learn how to fetch and display data from a backend API.
- **Component Design:** Build reusable, maintainable React components.
- **Error Handling:** Show loading and error states for better UX.
- **Responsive Layouts:** Use CSS and semantic HTML for mobile-friendly design.
- **Code Organization:** Structure code for clarity and scalability.

## Demo Script (for instructors)
1. Show the main page and explain the endpoints.
2. Perform a search and discuss the results.
3. Browse projects and show how pagination works.
4. Compare a design and highlight the analysis features.
5. Review the code structure and discuss best practices.
